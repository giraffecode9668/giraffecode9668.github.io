二、React技术

1、元素表示

元素是构成 React 应用的最小单位，它用于描述屏幕上输出的内容，有：const，let，var

三者区别：

  • const 在 ES6 中，const 只限制对应的变量不能够被再次赋值；但是变量的属性可以被再次赋值。
  • let 避免重复定义了变量，无意间修改了全局变量；是定义block local variable的。
  • var 非常灵活

参考 ：ES6 中 let, var, const 的区别
